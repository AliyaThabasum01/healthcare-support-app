# Healthcare Support Portal

## Overview

Healthcare Support Portal is a simple web application designed to help NGOs and healthcare organizations collect patient support requests digitally.

## Features

* Patient Support Form
* Name, Email, and Health Query Input
* AI-Inspired Query Categorization
* Automated Summary Generation
* Responsive User Interface

## Tech Stack

* HTML5
* CSS3
* JavaScript

## AI / Automation Idea

The application includes a simple AI-inspired feature that analyzes user queries and categorizes them automatically. For example:

* Fever → Fever Related Query
* Cough → Respiratory Related Query
* Headache → Neurological Related Query

This helps NGOs quickly understand and prioritize incoming requests.

## NGO Use Case

Non-governmental organizations often receive numerous support requests. This portal can be used to:

* Collect patient concerns online
* Organize requests efficiently
* Reduce manual effort through automated categorization
* Improve response time

## Installation

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in a web browser.
